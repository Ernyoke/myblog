# Myblog

This project is a tutorial on how to wire together a Spring project with an AngularJS frontend project.

## Build
Clone the project:
`git clone https://github.com/Ernyoke/myblog.git`

### Back-end
```
mvn clean install
```

### Front-end
(in `src\main\webapp\app` folder)
```
npm install
bower install
grunt build
```
